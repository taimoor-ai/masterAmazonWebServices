

# 🚀 COMPLETE AWS CLOUD ENGINEER ROADMAP (Top 10%)

## 🧠 Core Concepts (VERY IMPORTANT)

Don’t skip this — this is what separates average from top 10%

* What is **IaaS vs PaaS vs SaaS**
* **Regions, AZs, Edge locations**
* **High Availability vs Fault Tolerance**
* **Scaling (Vertical vs Horizontal)**
* **Load Balancing basics**
* **Networking basics (IP, DNS, Subnets, NAT)**

---

## ☁️ Key AWS Services

* EC2 (compute)
* S3 (storage)
* IAM (security)
* VPC (networking basics)
* CloudWatch (monitoring basics)

---

## 🛠 Hands-on Tasks

You MUST do these:

1. Launch EC2 manually
2. Host your React app on EC2
3. Store images/files in S3
4. Create IAM users/roles (NO root usage)
5. Setup basic VPC with:

   * Public subnet
   * Internet Gateway
