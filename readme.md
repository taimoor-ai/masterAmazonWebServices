### **Description:**

A personal learning repository to track progress in Amazon Web Services (AWS). This repo contains notes, tutorials, practice projects, and exercises from my ongoing AWS course. It serves as a hands-on reference for cloud computing concepts, AWS services, and real-world implementations.

### **Intro:**

This repository is organized to document my step-by-step learning path in AWS, including:

* Core AWS services (EC2, S3, Lambda, RDS, IAM, etc.)
* Hands-on labs and projects
* Configuration examples and scripts
* Key insights and best practices

The goal is to reinforce knowledge through practice and create a resource that can be revisited anytime.
